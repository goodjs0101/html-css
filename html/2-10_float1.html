float 속성은 자식 요소에 속성 값이 상속되지 않고 주위에 위치한 요소에 자동으로 속성 값이 상속되어 레이아웃에 영향을 끼친다. 이를 해제하기 위해서는 clear 속성을 이용해야 한다.

box1과 box2에 float 속성 값을 left로 지정하고 box3에는 float 속성을 지정하지 않을 경우 float 속성이 box3에 상속되어 다단을 이루어 레이아웃이 배치된다.

하지만 box3에 clear 속성을 지정하면 float 레이아웃을 해제하고 줄 바꿈을 하여 새로운 행으로 레이아웃을 배치하게 된다.

float: left / right
float 속성을 주면 원래 100%의 넓이가 자기 영역만큼 축소된다.
주위에 위치한 요소에 자동으로 속성 값이 상속되어 레이아웃에 영향을 끼친다.
box1과 box2에 float 속성 값을 left로 지정하고 box3에는 float 속성을 지정하지 않을 경우 float 속성이 box3에 상속되어 다단을 이루어 배치된다.
clear: left / right / both
상속되는 float 속성을 해제하기 위해서는 clear 속성을 이용해야 한다.
box3에 clear 속성을 지정하면 float 레이아웃을 해제하고 줄 바꿈을 하여 새로운 행으로 레이아웃을 배치하게 된다.
단, clear는 블럭 요소에만 적용된다.
